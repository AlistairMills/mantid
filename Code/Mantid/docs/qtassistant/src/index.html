Mantid Overview

The Mantid project provides a framework that supports high-performance computing and visualisation of scientific data.

Mantid has been created to manipulate and analyse Neutron and Muon scattering data, but could be applied to many other techniques.

The framework is open source and supported on multiple target platforms (Windows, Linux, Mac).
